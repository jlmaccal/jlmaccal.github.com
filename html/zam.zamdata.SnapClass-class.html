Class SnapClass

        SnapClass

Class that represents a snapshot.

A snapshot is a structure obtained after clustering the REMD simulations. Each snapshot contains a Pdb structure, parents, start and stop residues, and a list of contacts.

Note: By convention the variable for a snapshot is usally called Snap and a list of snapshots is usually Snaps.

Warning: The Score for a snapshot is initialized based on the number of contacts not on the logit map. Call UpdateScore to calculate score based on logit map.
