Class SampleRama

source code

object --+

        SampleRama

This class takes a sequence and returns a set of Ramachandran allowed backbone dihedral angles for it.

There are separate allowed dihedral maps for proline, glycine and all other residues. The algorithm samples using the quasi-random Sobol sequence, which is more uniform than a truly random sequence.

Initialize the sampling object.

The minimum loop length that can be sampled is 4. Note that LoopStart and LoopEnd follow the usual python convention where LoopStart is inclusive while LoopEnd is not. The loop to be sampled should include the hinge residues on either end, for which only the Psi (n-terminal hinge) or Phi (C-terminal hinge) torsions are sampled. The pivot residues are not sampled, which improves efficiency when doing loop closures as these residues are solved for anyway.

Parameters:
  • Protein (zam.protein.ProteinClass) - protein containing the loop to be sampled
  • LoopStart (int) - first residue of the loop (inclusive)
  • LoopEnd (int) - last residue of the loop (exclusive)
  • Pivots ((int, int, int) where the integers are the indices of of the pivot residues) - list of pivot residues for which sampling is not performed

GetTorsions(self)

source code 

Returns the next set of torsions.

Each time this function is called it will return a new set of Ramachandran allowed torsions using Sobol sampling. The output is an array of 3 x n_res torsions, where n_res includes the hinge residues. Only the Psi angle of the first hinge, the Phi angle of the last hinge, and the non-pivot interior phi and psi torsions will be sampled. All other torsions will be set to 180 degrees.

Returns: array
a set of 3 x n_res torsions
