Module pdbtools

source code

Tools to deal with pdb files

CONVENTION: Any command that takes Pdb as an argument can use either the pdb filename or the actual pdb data

CONVENTION: Residue and atom numbers start as 1 as in the Pdb file

GetOverlaps(Pdb, MinDist=None, FirstOnly=False)

source code 

Returns a list of tuples (a,b) of overlaps between atoms a and b. a and b start at zero and increment consecutively and are not taken from the pdb data.
