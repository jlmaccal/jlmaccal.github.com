GetDihCoordsFromStruct(protein)

source code 

This function returns the structure of a protein in internal coordinate.

Note that this routine only returns the phi,psi angles and thus, many of the internal coordinates (bond lengths, angles, etc) are missing. Also, residues without dihedrals will have values of zero.

Parameters:
  • p (ProteinClass) - structure to calculate coordinates for
Returns: numpy.array
a vector of dihedral angles

GetContactVectorFromStruct(protein)

source code 

Returns contact space represnetation of a protein structure.

This function maps a protein structure into contact space. The function _GetContactIndFromResPair is used to map a given pair of residues (i,j) into an index into contact space. contacts[k] is 1 if residues i,j are within 8 Angstroms of one another and 0 otherwise.

Parameters:
  • p (ProteinClass) - structure to calculate coordinates for
Returns: numpy.array
a vector of residue contacts
