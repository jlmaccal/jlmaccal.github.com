Class ZamDataClass

source code

object --+
         |
        ZamDataClass

Main ZAM database

This class is the central data structure in zam. Much of the information about a run is contained here, including the sequence, a list of restraints, and most importantly a list of fragments.

Note: There is usually one instance of this class called zd by convention.

AddRest(self, a, b, MinECO=3, Force=False)

source code 

Adds a restraint to the zam simulation.

The restraint is only added if it has an effective contact order of less than MinECO, unless Force == True.

Parameters:
  • a - index of residue a
  • b - index of residue b
  • MinECO - minimum effective contact order
  • Force - force addition of restraint even if contact order < MinECO

GetFrags(self, Stage=None, StartRes=None, StopRes=None, Length=None)

source code 

Returns a list of fragments, optionally with values in Stage or with given starting and/or stopping residues.

Typical usage:

 FragsInAssem = zd.GetFrags(Stage = "assem")
 FragsOfLengthTwelve = zd.GetFrags(Lentgh = 12)

By default returns all frags, same as zd.Frags

Parameters:
  • Stage (list of string) - list of stages to return.
  • StartRes (int) - return frags beginning with StartRes
  • StopRes (int) - return frags ending with StopRes
  • Lentgh - return frags Length residues long
  • Length (int)
Returns: list of zamdata.FragDataClass
all frags with specified properties
