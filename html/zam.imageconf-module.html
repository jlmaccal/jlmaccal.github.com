MakeImageConfs(PdbFiles, OutputPath='', Rotate=True, JustCartoon=False, Color=None, SizeX=100, SizeY=100)

source code 

Make images from a set of PDB files.

The color is specified as follows:

  • None: default coloring scheme.
    • Green: hydrophobic
    • Red: acidic
    • Blue: basic
  • Rainbow: pymol spectrum scheme
  • Spectrum: pymol spectrum scheme
  • Any other string: passed directly to pymol
Parameters:
  • PdbFiles (list of string) - a list of PDB files to make images for
  • OutputPath (string) - directory to put output files in - defaults to location of file
  • Rotate (bool) - flag to rotate so two highest principal components are parallel to the screen
  • JustCartoon (bool) - if False then include sidechain rendering
  • Color (None, string) - parameter controlling color.
