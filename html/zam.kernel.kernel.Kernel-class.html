build_from_condprob(self, condprobfile, zeroindex=20, AnchorCO=5, NumOffDiagonal=4)

source code  
Builds a kernel cmap from a conditional probability matrix.

REQUIRED ARGUMENTS
condprobfile   - the filename the the condprob cmap

OPTIONAL
zeroindex      - the index (i) of the matrix where the anchor (i,i+CO) is
                 For the single3465/*20x20.cmap file, zeroindex=20
AnchorCO       - the Contact Order for the anchoring  conditional contact.
                 Default is AnchorCO=5, the case for the single3465_05_20x20.cmap file.
NumOffDiagonal - the number off the (hairpin) diagonal to include.  The default
               - here is 4, for hairpins, helices should be 6 or 8.

logit_profile_bayes(self, logit_cmap, null_kernel_cmap, firstres=1)

source code  

Takes in a logit_cmap and the index of the first residue, and slides 
the kernel along the logit scores to compute a profile along the sequence.

The bayes score also needs a kernel_cmap p(null) to compare to, i.e.

    log [ p(c_i | beta)/p(c_i | null) ]

@param logit_cmap: contact map or logit map
@type  logit_cmap: numpy.array
@param null_kernel_cmap: contact map or logit map
@type  null_kernel_cmap: numpy.array
@param firstres: first residue to calculate profile for
@type  firstres: int
