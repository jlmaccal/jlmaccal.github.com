GetAllAlign(TarSeq, Seqs, MultiMap=False, MinLen=3, MaxTrim=None, MaximizeMaps=False)

source code 

Returns all alignments of Seqs to TarSeq.

@ param MultiMap: = True will allow each Seq to be mapped to TarSeq in multiple locations.

@ param MinLen: is the minimum number of residues that will be mapped.

@ param MaxTrim: is the maximum number of residues that will be trimmed from the ends of each seq in Seqs to do the alignment

@ param MaximizeMaps: True will only return alignments with the maximum number of alignments (i.e., minimal unaligned regions)
