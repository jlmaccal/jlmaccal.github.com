Class SnapDatabaseClass

        SnapDatabaseClass

Database of snapshots.

This is a database of SnapClass. When initialized with a zd, SnapDatabaseClass finds all of the snapshots of the Frags in zd and adds them to self. Each fragment is scored based on the logit map and redundant fragments can optionally be removed. The logit map is obtained by initializing a zam.zamcontact.ContactDatabaseClass object with the zd.

Typical usage:

   cd = zamcontact.ContactDatabaseClass(zd)
   sd = SnapDatabaseClass(zd, cd.Scores)

   for Snap in sd.Snaps:
       do_something_cool(Snap)

Note: by convention the snapshot database is usually called sd.

Initialize database.

Searches through all fragments and gets snaps. Contact scores are calculated if required and redundant snapshots are optionally removed.

ContactScores usually comes from cd.Scores, where cd is a zam.zamcontact.ContactDatabaseClass that has been initialized with the zd.
