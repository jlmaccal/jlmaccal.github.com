Class FragDataClass

source code

object --+
         |
        FragDataClass

Fragment data

This class contains much of the information from a zam run. This is the most complex data structure in all of zam. It represents almost all of the information known about a fragment from the sequence, run parameters, to the analyzed results of the REMD simulations.

Much of ZAM revolves around creating or modifying Frags, or performing simulation or analysis on them.

Note: By convention instances of FragDataClass are usually named Frag or some other name containing Frag, such as NewFrag. Frags is usually a list of FragDataClass instances.

To Do: This class is frighteningly big, we should refactor to make it more manageable.
