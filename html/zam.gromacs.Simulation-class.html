Class Simulation

source code

object --+
         |
        Simulation


This class performs a molecular dynamics simulation of minimization using GROMACS.

Main methods are:
    Running simulations:
        DoMD
        DoMin
    Adding restraints
        AddDistanceRestraint
        ClearDisanceRestraints
        AddTorsionRestraint
        ClearTorsionRestraints
        
Main data members:
    structure <- current structure of the system
    run_parameters <- change these to alter the run parameters (mdp options) of the system

Prepares a pdb file to run with GROMACS.

Runs pdb2gmx to create a new pdb file with appropriate atom names and puts the molecule in a rombic dodecahedron box with 1.2 nm of padding.
