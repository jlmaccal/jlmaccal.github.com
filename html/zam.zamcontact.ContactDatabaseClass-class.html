Class ContactDatabaseClass

source code

object --+
         |
        ContactDatabaseClass

Class to build and maintain a database of contact scores based on logistic regression model.

The most common usage is to initialize a new ContactDatabaseClass with a zd:

   cd = ContactDatabaseClass(zd)

This will create a new contact database, loop over all of the fragments in zd and build a logit table, which can then be accessed through:

   cd.Scores[x,y]
   cd.Counts[x,y]

You can also build helix and hairpin propensity profiles based on applying kernels to the logit map. See zam.kernel.kernel.:

   cd.HelixPropensity[index]
   cd.HairpinPropensity[index]
