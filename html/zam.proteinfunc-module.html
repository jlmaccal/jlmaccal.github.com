GetContactList(Pos1, Ind1=None, Pos2=None, Ind2=None, Radius=8.0, MinCO=3)

source code 

Returns a list of contacts for one or two position matrices. Ind1, Ind2 are list of indices for positions in Pos1, Pos2. Pos1, Pos2 are arrays of positions. Radius is the contact radius. MinCO is minimum contact order, only used when Pos2 is not present.
